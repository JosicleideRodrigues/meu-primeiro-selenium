 Automação com python e selenium

 Este repositório contém meus primeiros projetos de automação utilizando a biblioteca Selenium para interação com o navegador chrome.

 Projetos inclusos:
 1 Aberuta de site automática (test_exemplo.py) um script simples que inicializa o webdriver, abre um site específico e valida conexão.
 2 Robô de pesquisa no google (test_pesquisa.py) uma automação funcional que abre o google, localiza a barra de pesquisa, digita um termo de pesquisa e simula o clique na tecla "enter".
 Tecnologias utilizadas:
 - python
 - Selenium webdriver
 - Webdriver manager (para gestão automática dos drivers)
 - Git e github para versionamento

   O que aprendi:
   - - Configuração de ambiente virtual venv.
   - - Manipulação de elementos web por by.name.
   - - Fluxo de trabalho com git (add, commit, push).
   - - Resolução de conflitos de ramificações (main/master).
